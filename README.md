# AI Coding Workspace

A browser-based AI IDE concept that combines the editing experience of VS Code, model routing from OpenRouter, autonomous coding workflows inspired by Claude Code, and cloud workspaces similar to Replit Agent.

## Product pillars

- **AI Hub:** connect OpenAI, Anthropic, Google AI/Gemini, OpenRouter, Groq, DeepSeek, Mistral AI, and local Ollama models by adding an API key and selecting a model.
- **Monaco workspace:** VS Code-like editor with tabs, multi-file editing, split view, terminal, file explorer, search, Git panel, extensions, AI chat, and live preview.
- **Agent mode:** generate folders and files, edit code, run builds, fix errors, and commit Git changes from one prompt.
- **GitHub workflow:** clone repositories, create branches, commit, push, review pull requests, create PRs, and merge.
- **Live preview and deployment:** auto-reloading previews for React, Vue, Next.js, HTML, Node, Express, and Vite, then one-click deploys to Vercel, Netlify, or Cloudflare Workers.
- **Prompt and plugin marketplace:** templates for landing pages, dashboards, SaaS apps, chatbots, CRM/ERP, portfolios, ecommerce, plus integrations for GitHub, Docker, Supabase, Firebase, Vercel, Cloudflare, Figma, databases, and SSH.
- **Model comparison:** send the same prompt to multiple models and compare results side-by-side.
